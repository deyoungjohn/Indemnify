What Indemnify Does

Indemnify is an autonomous parametric risk mitigation and insurance underwriter designed explicitly for the machine-to-machine (M2M) economy. It intercepts outbound agent transactions, simulates execution via an off-chain MCP daemon, and scores risk. If a transaction fails or stalls, the on-chain settlement layer dynamically prorates partial cashouts based on bracketed risk matrices.
